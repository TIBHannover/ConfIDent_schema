    ConfIDent-schema

    This is a schema for the ConfIDent project that describes the necessary metadata requirements of academic events and event series.

    URI: https://tibhannover.github.io/ConfIDent_schema/

    Classes

    Class Description
    SchemaBasedThing A mixin used in classes that contain schema based values, such as the classifications used to denote the academic field of an event or the external identifiers used to denote a thing.
    NamedThing A mixin used to provide the attributes needed for the identification of a thing.
    AcademicEventSeries An academic event series describes the set of academic events which take place on a regular basis and thus have an established common identity. This identity is constituted, for example, through institutional continuity in the hosting of a series (e.g. by a specialised society), thematic focuses and/or a common label under which a series is defined (particularly name and acronym). Nevertheless, it is possible that each of these criteria may change over time.
    AcademicEvent An academic event is part of the established instruments of science communication as a format for conveying the latest scientific publications. It is defined by the meeting of researchers at a specific time and place (virtual or physical) and with a specific thematic focus to present, hear and discuss these publications. In contrast to other forms of events, academic events should primarily serve the exchange of results and methods of scientific research and their didactic mediation. Furthermore, a significant participation of scientific organizations in the realization of an academic event is constitutive for this type of event; for example, to distinguish it from events in which researchers mainly act as external experts or with a purely legitimising function.
    ProcessName A container for the various names denoting an academic event or series.
    ExternalIdentifier An identifer of an entity declared in another schema.
    DigitalObjectId A centrally registered identifier symbol used to uniquely identify digital objects given by International DOI Foundation.
    WikidataId The identifier of a thing (item) in Wikidata.
    GndId The identifier of a thing (item) in the German National authority file.
    TibkatId The identifier of a publication in the TIB catalog.
    DblpId The identifier of an academic event or series in dblp.
    WikiCfpId The identifier of an academic event or series in WikiCFP.
    EventFormatSpecification An academic event format specification is a plan specification that classifies a planned gathering of people in an academic context according to some sociocultural format, which provides implicit and explicit details on how this gathing is to be carried out by its participants in terms of achieving certain objectives, fulfiling certain conditions and performing certain actions. It thus concretizes the expectations of the contributors of an academic event. While the explicit details are often provied as concrete parts (e.g. deadline or attendance fee specifications), the implicit details depend on the semantics encoded in the words used for the classification of academic events (e.g. "conference" or "workshop"). Depending on the sociocultural background these classifications can overlap or vary to a certain degree.
    Deadline A container for event deadlines.
    Metric A container for statistical information about an academic event or academic event series.
    ProcessRelation A container for relations between academic events.
    Location A container for the information about the location in which an academic event takes place.
    City The city in which an academic event takes place.
    Country The country in which an academic event takes place.
    Region The region in which an academic event takes place. For non USA events this might often be negligible.
    Venue The venue at which an academic event takes place.
    AcademicField An academic field is the scientific subject of an academic event or academic event series according to some controlled vocabulary or thesaurus and as such requires the scheme URI.
    Context A container to provide extra information, such as call of papers event description.
    Contributor A contributor is a person or organization that holds a contributor role which is being realized in an academic event or academic event series.
    Sponsor A person or an organization whose role it is to sponsor an academic event or event series.
    Attendee A person whose only role it is to attend an academic event.
    Moderator A person that has the role to moderate an academic event.
    Reviewer A person that has the role to review the submissions of an academic event.
    Organizer An organizer of an academic event or event series.
    ContactPerson The contact person of an academic event or event series.
    CommitteeMember A members of an academic event committee.
    CommitteeChair The head of an academic event committee.
    Presenter A person that presents its work at an academic event.
    KeynoteSpeaker A 'keynote speaker' is a presenter that is an invited person - often a multiplier in his or her (research) field - responsible for delivering a keynote speech.
    Publication A published work, e.g. proceedings or conferenc paper, that is the output of an academic event or series.
    ConfIDentRecords A container to be able to bundle academic event data and series in one data file (e.g. YAML or JSON).

    Slots

    Slot Description
    type An abstract property that is reused in certain classes to differentiate their instances according to the type enums defined as the range.
    doi A property to provide a digital object identifier (DOI) according to https://doi.org/.
    gnd_id None
    wikidata_id A property to link an entity with its Wikidata identifier.
    name A property to provide a name of a schema entity.
    id A property to provide an internal id of a schema entity in the ConfIDent plattform.
    external_id A property to provide an external id of a schema entity.
    value a property to provide the literal value of a schema based entity.
    schema_name None
    schema_base_uri The base URI of the schema that provides the context for the schema based value.
    landing_page A property to provide the website to which a persistent identifier is resolving and which contains all the metadata about an academic event or series.
    organizers A property to provide the organizers of an academic event or series.
    academic_fields A property to describe the scientific subject(s) of an academic event or academic event series, according to some controlled vocabulary or thesaurus. If this is used, its subproperties [value] and [schema_name] are mandatory.
    website A property to provide the URL the official website of a academic event or academic event series.
    sponsors None
    publications The published works that are related to the academic event or event series, such as proceedings and video recordings of talks.
    topics A property to provide the theme and topics of an academic event or event series as free keywords, phrases or text.
    metrics A property to provide metrics of an academic event or series.
    context_info A property to provide extra information, such as call of papers event description.
    wikicfp_id A property to link a academic event or academic event series with its WikiCFP identifier.
    dpbl_id A property to link the a academic event or academic event series with its DBLP identifier.
    series_of A property to link to the events that are part of an academic event series.
    start_date The start date of an academic event or event series. Wheres the latter will in reality most likely be the start date of the first event of this series, unless there is some other source from which it is possible to derive the date of the inception of the series.
    end_date Similar to start_date but only for the end of an academic event or event series.
    event_status None
    in_series The relation used to provide the series of which an AcademicEvent is a part.
    event_format None
    at_location The location of the academic event.
    deadlines A property to provide deadlines of an academic event.
    related_to A property to be used
    ordinal The ordnial number of an academic event within its series.
    event_mode The event mode describes whether the event is a physical, virtual or hybrid event.
    official_name The official name of an academic event or series.
    acronym The official acronym of an academic event or series.
    former_name The former official name of an academic event or series. Usually this will only be needed in case an academic event series has undergone a name change.
    translated_name A translation of the official process name to be used in different language contexts.
    aliases A property to provide alternative names of an academic event or academic event series.
    specified_as A mandatory property to provide the format of an academic event as string, in order to further specify its type in case it could not be specified according to the possible values of EventType.
    deadline_date The date of a deadline.
    deadline_other A property to specify another type of deadline, if this type of deadline is not within the allowed values of DeadlineType.
    int_value A property to provide an integer value for a metric.
    str_value A property to provide a string value for a metric.
    rate_value A property to provide a rate value as float for a metric.
    truth_value A property to provide a truth value for a metric.
    description A property to provide a description of a metric.
    city The property to specify the City of an academic event location.
    country The property to specify the Country of an academic event location.
    region The property to specify the Region of an academic event location.
    venue The property to specify the Venue of an academic event location.
    lattitude The property to specify the lattitude of an academic event location.
    longitude The property to specify the longitude of an academic event location.
    meeting_url The property to specify the URL under which a one can participate virtually in an academic event.
    street The street of the venue including the house number seperated by comma.
    zip_code The zip code of the venue.
    text The free text used to provide more context information on an academic event or academic event series.
    license_str The license of the context information as text, which must be used when copying text from other sources.
    license_url The license of the context information as URL, which should be used when copying text from other sources and such a URL exists.
    contact The contact person of an academic event or event series.
    email The email address of the contact person.
    telephone The telephone number of the contact person.
    events A property to provide a list of academic events within this container.
    series A property to provide a list of academic event series within this container.

    Enums

    Enums Description
    EventType The most common minimal event types. For event types that are not in this list, you can use "other" and provide the label of this other event format using the event_format property.
    EventStatus The status of the academic event which indicates if it takes place as planned.
    EventMode An enum to specify if an academic event is in person, virtual or a hybrid of both.
    DeadlineType An enum that specifies the possible kinds of deadlines of an academic event.
    RelationType The kinds of relations that are allowed between academic events.
    MetricType The possible metric of an academic event.
    ContributorType An enumaration used to distinguish the contributors of an academic event or academic event series with regard to them being either an organization or a person.
